## Step-by-Step Instructions

### **Step 1: Create Replit Account**
1. Go to [replit.com](https://replit.com)

### **Step 2: Create New Replit Project**

**Option A: Upload from GitHub (Recommended)**
1. Go to [replit.com/new](https://replit.com/new)
2. Click **"Import from GitHub"**
3. Paste your GitHub repo URL
4. Click **"Import"**
5. Wait for import to complete

**Option B: Upload ZIP File**
1. Go to [replit.com/new](https://replit.com/new)
2. Click **"Import from GitHub"** â†’ (even if using ZIP, it shows the option)
3. Actually, click **"Create Repl"** â†’ Select **"Python"**
4. Drag & drop your project folder

**Option C: Manual Upload (Easiest)**
1. Go to [replit.com/new](https://replit.com/new)
2. Select **"Python"** as language
3. Name it: `Email-Spam-Detection`
4. Click **"Create Repl"**
5. In left panel, click **"Upload file"**
6. Upload these files/folders:
   - `app.py`
   - `Email_Spam_Detection_with_Machine_Learning.py`
   - `requirements.txt`
   - `templates/` folder
   - `static/` folder

---

### **Step 3: Install Dependencies**

**Replit will auto-detect `requirements.txt` and install packages**

If not automatic:
1. Open **Shell/Console** (bottom)
2. Run:
```bash
pip install -r requirements.txt
```

---

### **Step 4: Run Your App**

**Automatic (Replit):**
- Replit detects `.replit` file
- Click **"Run"** button (top)
- App starts automatically

**Manual:**
1. Open Shell/Console
2. Type:
```bash
python app.py
```

---

### **Step 5: Get Public URL**

After clicking Run:
1. You'll see a window open with your app
2. At top of the window: **Copy the URL**
3. Format: `https://your-repl-name.your-username.repl.co`

**Example:**
```
https://Email-Spam-Detection.YourUsername.repl.co
```

---

### **Step 6: Share Your App**

Your app is now **LIVE on the internet!**

**Share the URL:**
- With friends
- On social media
- In your portfolio
- In job applications
